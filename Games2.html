  • Programming Language: HTML, CSS, JavaScript
  • Development Environment:: Visual Studio Code
  • Project URL: Racing Game

Endless Speed: Master the Road

The game is a 2D racing game where the player controls a car on a randomly generated road, with the goal of staying on the road as long as possible. As the player progresses, the road becomes more challenging with tighter turns and direction changes. The player uses the "left" and "right" keys to steer, while "up" and "down" control the speed, affecting the car's handling. The randomly generated road makes each session unique, and speed must be carefully managed to avoid going off the road. Graphically, the game features a dynamic background that creates a sense of speed. White lines on the road mark the lanes, and the HUD displays the speed and direction. The physics simulation makes turns more difficult as speed increases. The game offers a smooth and challenging experience, with increasing challenges as the player progresses.
